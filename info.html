Información del proyecto

Referencioteca es un archivo vivo para observar, conectar y reconocer trayectorias en diseño.

Reunimos proyectos publicados de ex estudiantes de Diseño UC para facilitar su descubrimiento, estudio y difusión, con atribución y enlaces a las fuentes originales.

La plataforma crece de forma gradual. Puedes usar esta página como base editable para sumar equipo, metodología, hitos, criterios curatoriales y datos de contacto.

Qué es Referencioteca

Referencioteca es una plataforma de referencia y búsqueda curada orientada al diseño, donde cada proyecto mantiene autoría, año, área y vínculo de origen. El objetivo es visibilizar prácticas diversas, generar contexto y facilitar conexiones entre generaciones.

Puedes editar este bloque para afinar tono editorial, alcance temporal, criterios de selección y declaración pública del proyecto.

Cómo usar la plataforma

Vista Infinita
Exploración visual abierta para descubrir proyectos por relación temática, tags y similitudes formales.
Vista Grilla
Orden tipo masonry para lectura rápida de portafolio. Incluye botón aleatorio para reordenar sin recargar.
Vista Lista
Índice ordenable por proyecto, autor y año. Ideal para consultas comparativas o navegación directa.
Búsqueda
Admite múltiples palabras clave. Si no hay match estricto, aplica coincidencia parcial para evitar resultados vacíos.

Criterios editoriales y curaduría

Criterios mínimos

Autoría identificable, año aproximado, área o disciplina, y enlace público verificable. Se priorizan proyectos con contexto claro.

Fuentes

Sitios oficiales de autores/as, portafolios, publicaciones y redes públicas. Siempre con atribución y link al origen.

Actualización

Revisión periódica de links activos y mejora continua de metadatos. Se pueden corregir registros por solicitud.

Transparencia

Este espacio es editable para documentar cambios de criterio, límites de cobertura y decisiones curatoriales.

Preguntas frecuentes

¿Cómo propongo un proyecto?
Desde Submit puedes enviar los datos básicos para evaluación editorial.
¿Puedo pedir cambios?
Sí. Escríbenos con el detalle del ajuste y el enlace del proyecto para actualizar. Puedes realizarlo directamente en cada proyecto.
¿Puedo solicitar retiro?
Sí. Puedes pedir la edición o retiro escribiendo a referencioteca.cl@gmail.com o directamente en casilla del proyecto.
